# dotvim

My personal neovim setup.

```console
cd ~/.config
git clone -b nvim https://github.com/larsyencken/dotvim nvim
```

Then open neovim and run `:PlugInstall` to install plugins.

## Linting requirements

### Python

```
pip install flake8
```

### JSON

```
brew install node
npm install -g jsonlint
```
